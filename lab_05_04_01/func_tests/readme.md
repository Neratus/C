# Тесты для лабораторной работы №5 задание №4

## Входные данные
Двоичный файл с ключами выполнения программы
-sb сортировка студентов по фамилии, студенты с одинаковыми фамилиями упорядочиваются по имени
-fb вывод информации о студентах, фамилии которых начинаются с заданной подстроки, в другой файл
-db удалить всех студентов с успеваемостью ниже средней


## Позитивные тесты:
- 01 - Ключ sb, все фамилии одинаковы
- 02 - Ключ sb, файл упорядочен
- 03 - Ключ sb, файл неупорядочен
- 04 - Ключ sb, файл неупорядочен, 3 фамилии одинаковы
- 05 - Ключ sb, файл содержит 1 студента
- 06 - Ключ fb, файл содержит 1 студента с подходящей фамилией
- 07 - Ключ fb, файл содержит нескольких студентов с подходящей фамилией
- 08 - Ключ fb, файл содержит 1 студента с фамилией короче подстроки 
- 09 - Ключ fb, все студенты подходят условию
- 10 - Ключ db, в файле у всех студентов одинаковый средний балл
- 11 - Ключ db, в файле один студент
- 12 - Ключ db, в файле два студента с разными средними баллами




## Негативные тесты:
- 01 - Не передано ни одного аргумента
- 02 - Передан невалидный агрумент
- 03 - Не передано название файла
- 04 - Передан несуществующий файл
- 05 - В файле аргументов произвольная строка
- 06 - Ключ sb, в файле не содержатся данные о студентах
- 07 - Ключ sb, передан пустой файл
- 08 - Ключ fb, передан пустой файл
- 09 - Ключ fb, передана слишком длинная подстрока
- 10 - Ключ fb, в файле не содержатся данные о студентах
- 11 - Ключ fb, фамилий,подходящих под подстроку не существует
- 12 - Ключ fb, передана пустая подстрока
- 13 - Ключ db, в файле не содержатся данные о студентах
- 14 - Ключ db, передан пустой файл
